# AI Loan Eligibility System

## Overview

This project demonstrates a Rule-Based AI system for loan eligibility.

The system evaluates an applicant using predefined business rules.

## Features

- Input Validation
- Rule-Based Decision Engine
- Explainable AI
- Confidence Score
- Professional Report

## Project Structure

```
main.py
constants.py
validator.py
decision_engine.py
report.py
```
