# The Problem with Traditional Programming

Traditional Programming follows rules written by humans.

Example

Age >= 18

↓

Adult

Age < 18

↓

Minor

The computer simply follows instructions.

It never learns.

It never improves.

---

# A Bigger Problem

Imagine writing a program to identify a cat.

Can we write rules like this?

```
IF

Has 2 Eyes

AND

Has 4 Legs

AND

Has Tail

↓

CAT
```

No.

Because every cat looks different.

- Different color
- Different size
- Different pose
- Different lighting
- Different background

Writing millions of rules becomes impossible.

---

# Why AI Was Invented

Humans realized that some problems cannot be solved by writing rules.

Instead,

What if the computer could learn the rules by itself?

That idea became

**Artificial Intelligence.**

---

# Definition

Artificial Intelligence (AI) is the ability of a computer system to learn, reason, solve problems, and make decisions similar to humans.

Unlike traditional programming,

AI can improve its performance by learning from data.

---

# Traditional Programming vs AI

Traditional Programming

```
Human

↓

Writes Rules

↓

Computer Executes Rules

↓

Decision
```

Artificial Intelligence

```
Human

↓

Provides Data

↓

Computer Learns Rules

↓

Prediction / Decision
```
